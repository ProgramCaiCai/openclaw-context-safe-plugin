# Repository Guidelines

- README/docs wording: describe this as an independently implemented plugin.
- README/docs wording: do not mention a "fork" for this plugin unless the user explicitly asks for that background.
- README/docs wording: do not use the Chinese contrastive pattern `不是……而是……` unless the user explicitly asks for that phrasing.
- README install guidance: mention `scripts/install.py` as the primary one-command install path when documenting setup.
